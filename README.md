# To-do-Website

## Beschreibung
Kleine Web-App zur Verwaltung von To-Do-Aufgaben.  
Aufgaben können hinzugefügt, abgehakt und gelöscht werden.  
Daten werden im Browser (localStorage) gespeichert.  
Verwendete Schriftarten: **Nunito** und **sans-serif** (via Google Fonts).

## Technologien
- HTML
- CSS
- JavaScript
- Google Fonts: Nunito, sans-serif

## Features
- Hinzufügen neuer Aufgaben
- Aufgaben abhaken (Checkbox)
- Aufgaben löschen (mit Mülleimer-Icon)
- Aufgaben bleiben nach einem Reload erhalten (localStorage)
- Moderne Optik mit Hover- und Animationseffekten

## Icons
- Mülleimer- und Hinzufügen-Icons stammen aus **Font Awesome Free v7.1.0**  
  Lizenz: Free (https://fontawesome.com)

## Nutzung
- Einfach `index.html` im Browser öffnen
- Aufgaben hinzufügen, abhaken oder löschen
